Introduction

In the following module, you will learn to create two-dimensional objects using WebGL, intended as a foundation to design more complex graphics implementations using the framework.

First, you will learn about the WebGL context, including the implementation of shaders and buffers. A vertex shader and fragment shader are required anytime an image is drawn. A buffer is a fixed-sized chunk of memory used for efficient storage and access when processing streams of binary data in order to create a visual representation of an object.

Forms of Data accepted by Shaders

  • attributes - data derived from buffers
  • textures - data from pixels
  • uniforms - values that remain the same for all vertices of a given call to draw, such as an offset
  • varyings - data derived from vertex shader and interpolated

Types of Shaders

  • vertex shader - generates clip space coordinates. can accept attributes, textures, or uniforms
  • fragment shader - provides a color for the current pixel being rasterized. can accept textures, uniforms, or varyings
