Glossary

application programming interface (API) - a system of tools and resources enabling developers to create software applications

attribute - a runtime argument from other languages

buffer - a fixed-sized chunk of memory used for efficient storage and access when processing streams of binary data

cascading style sheet (CSS) - used to format the layout of HTML web pages defining uniform styling for text, tables, etc. in a single instance for multiple pages by reference.

constant (const) - a mutable variable declaration preventing the variable identifier from being reassigned

compilation - translation process in which an executable machine code version of a program written in an interpreted language such as JavaScript is generated by compiler to be used in subsequent runs

Document Object Model (DOM) - an API for HTML documents that provides a representation of the page and a logical framework for access and manipulation

Element object - represents an HTML element, like paragraph (P), generic container for flow content (DIV), hyperlink (A), TABLE, etc.

executable - software (programs or operating information) in a form that can be run by computer

floating point numbers (float) - a numerical data type with a decimal allowing for higher precision (up to 24 bits, roughly 7 decimal places)

fragment shaders - inform WebGL about how the shapes composed by WebGL should be rendered, such as what color items should be

GL shading language (GLSL) - a C-like language designed specifically for creating shaders. In WebGL, GLSL code can be embedded into JavaScript as a string

graphics processing unit (GPU) - a supplementary processor containing hundreds of smaller cores allowing for parallel processing of intensive processes

hypertext markup language (HTML) - a universal scripting language to create webpages utilizing hyperlinks and tags

if - keyword as part of a statement specifying a block of code to be executed, if a specified condition is true

keyword - a reserved word utilizing a special form of syntax

machine code - bytecode interpretable by the computer

mat4 - 4x4 matrices of floating point numbers

method - action that can be performed on objects

mutable - a changable object which can be modified after creation

null a value representing the intentional absence of any object value

operand - specifes and represents data to be manipulated

operator - performs some operation on single or multiple operands (data value) producing a result

return - keyword used to exit subroutine, resuming instructions immediately after subroutine, sometimes part of a return statement including a return value or other information from subroutine

scalar - a single number or value represented by either an integer, floating-point, or string

shader - specialized functions written in GLSL language that run on GPU

string - an array of encoded characters, specified with quotation marks (“”)

syntax - set of rules defining the structure of expressions or statements in a programming language

transformation - a rule set systematically applied to values change size, position, or orientation

uniform - global GLSL variable requiring a value property passed to shader programs used to describe, calculate, and render visual effects

vec4 - a datatype consisting of a floating point vector comprised of four components (scalars or vectors)

vector - one-dimensional array

vertex shader - common form of three-dimensional shader that runs a single time per vertex, used for transform position and define properties such as color, texture

void - a unary operator keyword (appearing before a single operand) usable with any data type, specifying an expression to be evaluated without returning a value
