Drawing With Points in WebGL: Draw an Arc

Exercise

For this activity, you'll continue to practice simple WebGL drawings using vertex points. Your goal is to alter the code in the Assisted Code tab to create an arc. If you get stuck, you can peek at our implementation by clicking on the Solution tab.

The equation of circle is X2 + Y2 = r2, where r is radius.
